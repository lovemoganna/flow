1 Captura-cli¶

@echo off
set captura-cli="%ProgramFiles(x86)%\Captura\captura-cli.exe"

1.1 Captura-cli的两种模式:¶

  1. 当录制长度被指定的时候,录制会一直运行直至到指定的长度。

  2. 当录制长度没被指定的时候，按下q就会退出录制。

1.2 参数列表¶

参数

描述

–cursor

包括鼠标

–keys

包括按键

–clicks

包括鼠标点击

–delay

开始录制前的延迟（以毫秒为单位）

-t or –length

录制时长（单位: 秒）

–source

录制的位置 (See 1.6 开始位置截图参数 - verb source)

–mic

要使用的麦克风索引.(-1 = 无（默认))（0 是第一个设备)。

–speaker

要使用的扬声器输出索引。 （-1 = 无（默认））（0 是第一个设备）。

–webcam

要使用的网络摄像头。 （-1 = 无（默认））（0 是第一个网络摄像头）。

-f or –framerate

帧率（默认为 10）

–encoder

要使用的视频编码器。见 1.4 开始录制参数 - verb start

–vq

视频质量(1 到 100)(默认为 70)

–aq

音频质量（1 到 100）（默认为 50）

-f or –file

输出文件路径

-y

覆盖现有文件

–replay

回放录制.以秒为单位指定持续时间作为参数。例如: –replay 20

1.3 使用示例¶

1.3.1 使用光标和按键以及来自第一个扬声器输出的音频录制10秒¶

Target

  • 光标

  • 按键

  • 第一个扬声器输出的音频

  • 录制10秒.

  • Code 示例:

set captura="%ProgramFiles(x86)%\Captura\captura-cli.exe"
start "" %captura% start --length 10 --cursor --keys --speaker=0

1.3.2 使用其录制固定窗口¶

1.录制固定窗口 2.绑定键盘开始 / 取消 / 暂定 3.保存位置 4.编码器 5.制作封面

DIM objShell
set objShell=wscript.createObject("wscript.shell")
iReturn=objShell.Run("record-special-window.bat", 0, TRUE)

(defalias 'sc 'shell-command)
(defun video-record()
  "在当前文件夹中生成GIF图."
  (interactive)
  (sc "%AppData%/.emacs.d/.utils/dos/record-special-window.bat")
  (message "video已经生成!")
  )

1.4 开始录制参数 - verb start¶

默认情况下，使用 SharpAvi Motion JPEG 编码器。

1.4.1 SharpAvi¶

使用 sharpavi:<index> 作为参数. index是标识编码器的从零开始的索引.

您可以使用 captura-cli list 检查编码器索引。

  • 视频大小: 13.3M,质量高,体积大,下乘.

1.6 开始位置截图参数 - verb source¶

1.6.1 Desktop¶

使用 desktop 参数来捕获整个桌面(默认).适用于 captura-cli start 和 captura-cli shot.这是默认选项,因此与不使用此选项一样.

(("Microsoft" "Windows" "[Version" "10.0.22000.120]" "" "") ("(c)" "Microsoft" "Corporation." "All" "rights" "reserved.") ("" "" "" "" "" "") ("c:\\Users\\admin\\Desktop\\html\\sphinx-blog\\docs\\source\\Windows\\captura>@echo" "off" "" "" "" "") ("More?" "" "" "" "" ""))
%captura-cli% start --source desktop

1.6.2 Region¶

分别使用左侧、顶部、宽度和高度.作为逗号分隔值来表示要捕获的区域.适用于 captura-cli start 和 captura-cli shot. 该区域的尺寸必须是均匀的。 如果不是，则根据需要将它们减 1.

以选择区域截图为例:

(("Microsoft" "Windows" "[Version" "10.0.22000.120]" "" "") ("(c)" "Microsoft" "Corporation." "All" "rights" "reserved.") ("" "" "" "" "" "") ("c:\\Users\\admin\\Desktop\\html\\sphinx-blog\\docs\\source\\Windows\\captura>@echo" "off" "" "" "" "") ("More?" "" "" "" "" ""))
%captura-cli% shot --source 0,0,960,1080

1.6.3 Screen¶

使用 screen:<index> 作为参数. index 是标识屏幕的从零开始的索引. 适 用于 captura-cli start 和 captura-cli shot.

你可以使用 captura-cli list 检查屏幕索引.结果为:

AVAILABLE SCREENS
------------------------------
0 : \\.\DISPLAY6

1.6.4 No Video¶

如果不想录制视频使用 none. 仅适用于 captura-cli start.可用于仅音频录制,例 如仅记录扬声器输出:

(("Microsoft" "Windows" "[Version" "10.0.22000.120]" "" "") ("(c)" "Microsoft" "Corporation." "All" "rights" "reserved.") ("" "" "" "" "" "") ("c:\\Users\\admin\\Desktop\\html\\sphinx-blog\\docs\\source\\Windows\\captura>@echo" "off" "" "" "" "") ("More?" "" "" "" "" ""))

你可以使用 captura-cli list 检查可见的窗口句柄.
